# Installation

## Stable release

To install MovFX, run this command in your terminal:

```bash
$ pip install movfx
```

This is the preferred method to install MovFX, as it will always install the
most recent stable release.

If you don't have [pip](https://pip.pypa.io) installed, this
[Python installation guide](http://docs.python-guide.org/en/latest/starting/installation/)
can guide you through the process.

## From sources

The sources for MovFX can be downloaded from the
[Github repo](https://github.com/mediatoolbox-org/movfx).

You can either clone the public repository:

```bash
$ git clone https://github.com/mediatoolbox-org/movfx
```

Or download the
[tarball](https://github.com/mediatoolbox-org/movfx/tarball/main):

```bash
$ curl -OJL https://github.com/mediatoolbox-org/movfx/tarball/main
```

Once you have a copy of the source, you can install it with:

```bash
$ poetry install
```
